# Love Heart Site

Анимированный React + Vite + TypeScript проект с сердцем, фото и сообщениями.

## Скрипты

- `npm run dev` — запуск dev-сервера
- `npm run build` — сборка проекта
- `npm run preview` — предпросмотр production-сборки
- `npm run lint` — проверка кода ESLint

## Старт

Установите зависимости и запустите проект:

```bash
npm install
npm run dev
```

Если зависимости уже установлены, для экрана нужен пакет `framer-motion`.
